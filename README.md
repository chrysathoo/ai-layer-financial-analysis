## 📋 專案簡介

本專案提供一套系統化的 **AI產業四層財務分析框架**，用以追蹤 AI 價值鏈各層的收入、利潤、資本支出與資金流向。

對應黃仁勳提出的「AI五層蛋糕論」（Energy → Chips → Infrastructure → Models → Applications），本框架將產業拆分為：

- **Layer 0**：基礎設施層（能源、晶片製造、設備）
- **Layer 1**：算力層（NVIDIA、AMD、Oracle OCI、Hyperscaler 等）
- **Layer 2**：模型層（OpenAI、Anthropic 等）
- **Layer 3**：應用層（SaaS、物理 AI 等）

重點追蹤各層毛利率/營業利益率、CapEx 執行情況、跨層資金流分配，以及關鍵數據缺口。

---

## 🎯 用途

- 支撐 AI 產業供應鏈與需求真實性分析
- 提供高可信度（✅ SEC確認）財務數據整理
- 作為長期追蹤 Regime Shift 的原始數據來源
- 幫助投資人、產業分析師理解 AI 價值鏈的利潤分配結構
